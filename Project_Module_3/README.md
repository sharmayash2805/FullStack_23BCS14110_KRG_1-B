# Project Module 3: Product Management & Catalog

**Phase:** 3 - Product Inventory and Catalog System

## Overview
- Product database schema and CRUD operations
- Product listing and filtering
- Product search functionality
- Inventory management

## Structure
```
├── Backend/           Product management APIs
├── Frontend/          Product listing and search UI
└── README.md          This file
```

## Quick Start

### Backend
```bash
cd Backend
mvn clean install
mvn spring-boot:run
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```

## Key Features
- Product CRUD endpoints
- Advanced product filtering
- Full-text search capability
- Inventory tracking
- Product categorization
- Image upload for products
