# Project Module 4: Shopping Cart & Orders

**Phase:** 4 - Shopping Cart and Order Processing System

## Overview
- Shopping cart management
- Order creation and processing
- Payment gateway integration
- Order history and tracking

## Structure
```
├── Backend/           Cart and order APIs
├── Frontend/          Cart and checkout UI
└── README.md          This file
```

## Quick Start

### Backend
```bash
cd Backend
mvn clean install
mvn spring-boot:run
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```

## Key Features
- Add/remove items from cart
- Cart total calculation
- Order creation endpoint
- Payment processing
- Order status tracking
- Order history retrieval
- Invoice generation
