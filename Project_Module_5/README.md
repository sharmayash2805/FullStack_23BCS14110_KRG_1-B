- Admin dashboard and management panel
- Analytics and reporting
- User management interface
- System monitoring and logs

## Structure
```
├── Backend/           Admin and analytics APIs

```

## Quick Start

### Backend
```bash
cd Backend
mvn clean install
mvn spring-boot:run
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```

- Admin login and authentication
